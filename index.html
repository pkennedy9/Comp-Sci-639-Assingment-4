Visualizing Life Expectancy

This visualization consits of a dropdown box and an interactive map where the user can select different features that they wish to see presented on the map. Once you select a feature, the map will change to be linked with that feature. The different shades of each country represents its feature value - with higher values corresponding with a darker shade of blue. As the user hovers over each country, the feature value will populate. There are seven features to choose from: Life Expectancy, Diphtheria immunization, Hepatitis B immunization, Polio immunization, average BMI, and GDP.

Note: Diphtheria immunization, Hepatitis B immunization, and Polio immunization are expressed as a percentage out of 100.



This graph plots the correlation between the selected feature and life expectancy. We included this plot to see if there are any inights that can help show how life expectancy relates to the different variables above.
